Using Stable Diffusion

These sets of images were created using an AI image generator called Stable Diffusion. The generator works by analyzing a prompt (a sentence or collection of words) as reference and scowers the internet for images with tags related to the words in the prompt. It then takes certain elements from the collection images and combines them together into a single image. That's basically a simplified explanation of how Stable Diffusion works.
